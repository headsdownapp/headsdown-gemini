# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-04-25
### Added
- Specialist `@headsdown` subagent for availability management.
- `BeforeAgent` hook for real-time focus warnings.
- Interactive `ask_user` UI for focus conflict resolution.
- Secure token storage via manifest settings.
- Automated GitHub Release creation.

### Changed
- Modernized GitHub Actions to v6 and Node 24.
- Updated `@headsdown/sdk` to 0.2.14.
- Simplified extension manifest with `${extensionPath}`.
